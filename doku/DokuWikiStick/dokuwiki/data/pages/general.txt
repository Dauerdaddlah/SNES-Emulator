Es gibt zwei Versionen vom SNES
  * NTSC (Amerika)
    * Masterclock 21,477 MHz bzw. 21477000 Hz
    * CPU 3,579545 MHz bzw. 3579545 Hz
    * 60 Hz
    * 256 x 224 px (width x height)
  * PAL (Deutschland)
    * Masterclock 21,28137 MHz bzw. 21281370 Hz
    * CPU 3,546895 MHz bzw. 3546895 Hz
    * 50 Hz
    * 256 * 239 px (width x height)
